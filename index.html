Politique de Confidentialité - Easy Daily

Dernière mise à jour : 24 mai 2024

1. Collecte des données

Easy Daily est une application hors-ligne par défaut. Nous ne collectons, ne stockons, ni ne partageons aucune de vos données personnelles sur des serveurs externes.

2. Stockage des données (SQLite)

Toutes les données que vous créez (plats, menus, photos) sont stockées localement sur votre appareil via une base de données SQLite. Ces données ne quittent jamais votre téléphone.

3. Achats In-App

Les transactions liées aux fonctionnalités Premium sont gérées exclusivement par le Google Play Store ou l'Apple App Store. Nous n'avons pas accès à vos informations bancaires.

4. Permissions

L'application peut demander l'accès à votre galerie ou caméra uniquement pour vous permettre d'ajouter des photos à vos plats. Ces images restent sur votre appareil.

5. Suppression des données

Si vous désinstallez l'application, toutes les données locales (menus, plats) seront définitivement supprimées de votre appareil.

Pour toute question : votre-email@exemple.com
